### 装饰器（函数作为参数时候的闭包）
将函数封装成装饰器函数，可以依次封装多次调用，避免重复代码  
```python
def timer(function): # 装饰函数声明
    def inner(a, b):
        print("start")
        function(a, b)
        print("stop")
        return 0
    return inner
    
@timer # 语法糖
def sum_diy(a, b) # 被装饰函数声明
    print('{}+{}={}'.format(a, b, a + b))
    return 0

# 函数调用,带参数
sum_diy(a, b) 
```
装饰器可以带参数  
```python
def new_timer(argv):
    def timer(func):
        def inner(a, b):
            print('start')
            funct(a, b)
            print('stop')
            return 0
        return inner
    return timer
    
@new_timer('add')
def test_sum(a, b):
    print('{}+{}={}'.format(a, b, a + b))
    
test_sum(1, 2)
```
